# HangMan Game

A fun and interactive Hangman game built with HTML, CSS, and JavaScript, complete with difficulty levels, visual feedback, emoji effects, and sound-based celebrations!

How to Play

1. Enter your name.
2. Select levels(Easy, Medium, and Hard).
3. Click Start Game.
4. Guess the letters/word according to the question or hint.
5. Press the on-screen keyboard to guess the letter.
6. Get the word right to increase your score.
7. Game ends after 6 incorrect guesses.

Features

- Difficulty Levels: Easy, Medium, Hard

- Player Name Input

- Dynamic Questions with Clues

- Score Tracking and High Score Persistence

- Visual Feedback with Hangman Images

- Sound Effects for Achievements

- Emoji Animations for Celebrations

- Responsive Design

Technologies Used
- HTML5

- CSS3

- JavaScript

- Audio (<audio> tag)

- Animations (@keyframes for emojis and alerts)

- LocalStorage (for high score)

Custom Features Highlight

- Animated Emoji Burst – celebratory animations using 🎉 and 🎊 emojis.

- Custom Pop-up Alerts – alert box shows up with animation on a high score.

- Stickman Progress Image – image updates as wrong guesses increase.

Installation

1. Clone this repository:
    - https://github.com/rozeensubedi/HangMan.git

2.  Add your own images and sounds:
    - Add hangman image files to the /img folder.
    - Add the applause sound file to the /sound folder.
  
3.  Open index.html in your browser to start playing!
